# 🌿 Mirana Intelligence — Money Tracker Bot

An advanced multi-chain anomaly radar and large transaction infrastructure custom-built for the **Mantle Network** ecosystem. Developed for **The Turing Test 2026 Hackathon** under the *AI Alpha & Data Track*, proudly sponsored by **Mirana Ventures** and powered by **Surf AI Core**.

## 🚀 Project Overview

**Mirana Intelligence** bridges the gap between Centralized Exchange (CEX) liquidity movements and Decentralized On-Chain (DEX) tracking. By combining real-time trade telemetry streams with predictive generative analysis, the platform provides institutional-grade whale surveillance for Mantle ($MNT) and major market assets ($BTC, $ETH, $SOL, $BNB).

## ⚡ Core Innovation & Features

* **🔗 On-Chain Explorer Node:** Direct integration with Mantle Network RPC indexers to extract, verify, and broadcast real-time transaction hashes with precise block mapping.
* **📊 Bybit V5 Live Telemetry:** Low-latency trade stream connection utilizing the Bybit V5 spot engine to monitor aggressive order book execution and institutional whale accumulation.
* **🧠 Surf AI Macro Analytics:** Generative market analysis pipeline that evaluates multi-coin data matrices using Surf AI models to produce instant, actionable sentiment overviews.
* **🛡️ Industrial-Grade Resiliency:** Integrated **Automatic Failover Cache Layer** that gracefully switches to mirrored secure historical transaction states if public blockchain indexers encounter rate-limits or HTTP 502/Timeout spikes, ensuring 100% platform uptime during volatile market events.

---

## 🛠️ Architecture & Tech Stack

* **Backend Core:** Python 3.10+ (Asynchronous Event Loop, Multithreading)
* **Interface Layer:** Telebot Engine (Telegram Interactive Callbacks & Inline Matrix Keyboards)
* **Database Ledger:** SQLite3 (Local high-speed deduplication indexer)
* **AI Analytics:** Surf AI Engine API
* **Data Aggregator:** Bybit V5 Market Ticker & Live Execution API
* **Presentation Hub:** HTML5, Tailwind CSS v4, tsParticles Dynamic Canvas with Web-Audio Controller
